# BlankFramework

#### 介绍
为了学习和了解编程思想，决定开始的一个写一个mvc框架
```
1.本框架意义在于理解IOC容器，以及依赖注入的思想
2.本框架富含基本容器以及依赖注入
3.框架具有路由以及中间件和基本orm功能

```

#### 软件架构
初步建设小型容器，满足基础的mvc服务，如何写出的会以文章形式公布


#### 安装教程

```
composer install
```
